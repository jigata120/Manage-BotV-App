Quick Start Guide

Script Integration

<script src="http://127.0.0.1:8080/static/js/integration/embed-chatbot.js"
        data-domain="http://127.0.0.1:8080"
        data-width="100%"
        data-height="100vh"
        data-theme="dark">
</script>

Place the script just before the closing </body> tag for optimal performance.

Configuration Options

Parameter Description Example
data-domain Sets the domain for the chatbot data-domain="http://127.0.0.1:8080"
data-width Controls the chatbot's width data-width="100%"
data-height Controls the height of the chatbot window data-height="100vh"
data-theme Sets the color theme data-theme="dark"
